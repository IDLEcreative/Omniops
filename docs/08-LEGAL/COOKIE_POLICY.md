## 1. What Are Cookies

### 1.1 Definition
Cookies are small text files that are placed on your device (computer, smartphone, tablet) when you visit a website. They help websites remember information about your visit, making your next visit easier and the site more useful to you.

### 1.2 Similar Technologies
In addition to cookies, we may use:
- **Local Storage**: Stores data in your browser with no expiration
- **Session Storage**: Stores data for the duration of your browser session
- **Web Beacons**: Small graphics that track usage
- **Device Fingerprinting**: Technical information about your device
- **ETags**: Cache validators used by browsers

### 1.3 First-Party vs Third-Party
- **First-Party Cookies**: Set by OmniOps directly
- **Third-Party Cookies**: Set by our service providers or partners

## 3. Types of Cookies We Use

### 3.1 Essential Cookies

These cookies are necessary for the Service to function and cannot be disabled.

| Cookie Name | Purpose | Duration | Data Collected |
|------------|---------|----------|----------------|
| `session_id` | Maintains user session | Session | Session identifier |
| `auth_token` | Authentication token | 7 days | Encrypted auth token |
| `csrf_token` | Security against CSRF attacks | Session | Security token |
| `load_balancer` | Routes traffic to correct server | Session | Server identifier |
| `consent_given` | Records cookie consent | 1 year | Consent status |

| `sidebar_state` | Dashboard layout preference | 30 days | UI state |

**Technical Implementation Notes for Developers:**

### Setting Cookies
```javascript
// Essential cookie example
document.cookie = "session_id=abc123; Secure; HttpOnly; SameSite=Strict";

// Preference cookie with expiry
document.cookie = "theme=dark; max-age=31536000; Secure; SameSite=Lax";
```

### Cookie Attributes Used
- `Secure`: Only sent over HTTPS
- `HttpOnly`: Not accessible via JavaScript
- `SameSite`: CSRF protection
- `Domain`: Scope limitation
- `Path`: URL path restriction

### Widget Implementation
```javascript
// Widget cookie configuration
window.ChatWidgetConfig = {
  cookies: {
    enabled: true,
    sameSite: 'none',  // For cross-origin embedding
    secure: true,      // HTTPS only
    prefix: 'omniops_' // Namespace cookies
  }
};
```
