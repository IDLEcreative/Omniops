Accessibility Note

All animations automatically disable if you have "Reduce motion" enabled in your system settings. Test this by enabling reduced motion in your OS accessibility settings and refreshing this page.

How to test:
• macOS: System Preferences → Accessibility → Display → Reduce motion
• Windows: Settings → Ease of Access → Display → Show animations
• Chrome DevTools: Cmd+Shift+P → "Emulate CSS prefers-reduced-motion"

Implementation Example

<script>
  window.ChatWidgetConfig = {
    serverUrl: 'https://your-domain.com',
    appearance: {
      showPulseAnimation: true,  // Enable/disable pulse ring
      showNotificationBadge: true, // Enable/disable green dot
      startMinimized: true,        // Start compact or expanded
    },
  };
</script>
<script src="https://your-domain.com/embed.js"></script>
